Notepad Extensions 1.2

About

	Extends Windows Notepad with a few additional features.

	Written by Brenton Gunning. http://brentongunning.com.

Features

	+ Autosave
	+ CTRL+W to quit
	+ Tab alignment
	+ Open from system tray

Notes

	The 32-bit version of Notepad Extensions only works with the 32-bit version of Notepad.
	The 64-bit version of Notepad Extensions only works with the 64-bit version of Notepad.
